# Introducción a SysrotCore: El CLI del Futuro

En el vertiginoso mundo del desarrollo web de **2025**, crear aplicaciones modernas requiere más que solo conocimiento técnico. Necesitas herramientas que aceleren tu flujo de trabajo sin comprometer la calidad. **SysrotCore** nació precisamente para resolver este desafío.

## 🚀 ¿Qué es SysrotCore?

SysrotCore es un **CLI de nueva generación** que te permite crear proyectos Next.js 14+ completamente configurados en menos de 30 segundos. Pero no es solo otro generador de boilerplate: es un ecosistema completo de desarrollo.

```bash
# Un solo comando para crear una aplicación completa
npx sysrotcore mi-app-increible
```

## ✨ Características Revolucionarias

### 1. Inteligencia Artificial Integrada

Por primera vez en un CLI, SysrotCore incluye **3 modelos de IA** listos para usar:

- **GPT-4o** - Para razonamiento avanzado y generación de código
- **Claude 3.5 Sonnet** - Análisis profundo y documentación inteligente  
- **Gemini Flash Pro** - Procesamiento multimodal ultrarrápido

```typescript
// Ejemplo: Chat con múltiples modelos de IA
import { useAI } from '@/hooks/useAI';

export function ChatComponent() {
  const { sendMessage, messages, isLoading } = useAI();
  
  const handleSubmit = async (message: string) => {
    // Automáticamente usa el mejor modelo según el contexto
    await sendMessage(message);
  };
  
  return (
    <div className="chat-container">
      {/* Interfaz unificada para todos los modelos */}
    </div>
  );
}
```
